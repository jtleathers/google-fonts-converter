Convert Google Fonts URL to use Variable Font Files

Convert your Google Fonts URL to use the variable font files instead. This will only work if the Google Font has variable fonts.

You can read more about variable fonts on web.dev or Google Fonts.

Note: This currently only supports font weights and styles (normal and italic). Features like Optical Size are not currently supported. Feel free to contribute to make that change though!

Examples

Multiple Font Familes with Italic and Normal

Original URL:
https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300;0,400;0,500;0,800;1,300;1,400;1,500;1,800&family=Playfair+Display:ital,wght@0,400;0,700;1,400;1,700&display=swap

Updated URL:
https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300..800;1,300..800&family=Playfair+Display:ital,wght@0,400..700;1,400..700&display=swap
