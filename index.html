Convert Google Fonts URL to use Variable Font Files

This converter takes a normal Google Fonts URL and converts it into a Google Fonts URL that will return variable font files instead. You can read more about variable fonts on web.dev or Google Fonts.

Notes: Make sure you only use Google Fonts that are classified as variable fonts. This converter only supports font weights and styles (normal and italic) so features like optical size are not currently supported. Feel free to contribute to make that change though!

Examples

Below are examples of the original URL provided by Google and how the converter will change it to serve variable fonts instead.

Multiple Font Familes with Italic and Normal

Original URL

https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300;0,400;0,500;0,800;1,300;1,400;1,500;1,800&family=Playfair+Display:ital,wght@0,400;0,700;1,400;1,700&display=swap

Updated URL

https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300..800;1,300..800&family=Playfair+Display:ital,wght@0,400..700;1,400..700&display=swap

Multiple Font Familes with only Normal

Original URL

https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;500;700&family=Playfair+Display:wght@400;500;700&display=swap

Updated URL

https://fonts.googleapis.com/css2?family=Open+Sans:wght@300..700&family=Playfair+Display:wght@400..700&display=swap

One Font Family with Italic and Normal

Original URL

https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300;0,400;0,500;0,800;1,300;1,400;1,500;1,800&display=swap

Updated URL

https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300..800;1,300..800&display=swap

One Font Family with only Normal

Original URL

https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;500;800&display=swap

Updated URL

https://fonts.googleapis.com/css2?family=Open+Sans:wght@300..800&display=swap
